== Installation ==

1. Upload the `one-click-start` folder to the `/wp-content/plugins/` directory.  
2. Activate the plugin from the **Plugins** menu in WordPress.  
3. Open **One Click Start** from the admin sidebar to create or import a setup recipe.

== Frequently Asked Questions ==

= Can this break my site? =  
The plugin is designed with safety in mind. Tasks run in controlled batches to prevent crashes or timeouts.  
Still, always use it on a fresh installation or create a backup first.

= Can I install my own plugins or themes? =  
The free version includes a curated list.  
**Pro version** adds full repository search and premium ZIP uploads.

= Does it support Multisite? =  
Currently optimized for single-site installations.  
Full Multisite compatibility may come in a future release.

= How does Import/Export work? =  
Export saves your configuration as a `.json` recipe.  
Import instantly loads the same setup on another WordPress site.

= Can I uninstall the plugin after running it? =  
Yes. All installed plugins, themes, and settings remain active even after uninstalling One Click Start.
